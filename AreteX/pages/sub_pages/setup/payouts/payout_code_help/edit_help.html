Edit Contributor Payout Template

Note: Changes to this template will NOT change products that have already used this template. Template parameters are stored on the product, and not dynamically read from the template.

Payment Type - This tells contributors why they are being paid. For example a Royalty might be appropriate for an author. An Instructor's Fee might be appropriate for a teacher.

Amount - This number is either the dollar amount per sale, or the percentage of sale, depending on "Paid On" value.

Paid On

  • If Fixed Amount - Amount is the number of dollars that will be paid each sale.
  • If Percent Gross - Amount will be used to compute payment as a percentage of the gross of the price of the product.
  • If Percent Net - Amount will be used to compute payment as a percentage of the net after expenses.
